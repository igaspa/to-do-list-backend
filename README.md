## Prerequirements and installation
- Git
- NodeJS

Clone this repository

```
git clone https://github.com/igaspa/to-do-list-backend.git
```

Installing dependencies

```
npm install
```

Starting the app

```
node server.js
```

Open http://localhost:3000 to view it in the browser.
